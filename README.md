# 📄 Resume Matcher & AI Generator

Choosing the right resume for a specific job description can be frustrating. This tool eliminates the guesswork by analyzing your existing resumes against a Job Description (JD) and automatically determining the best fit. If neither resume is a strong match, it builds a brand-new, ATS-optimized resume for you.

## ✨ Features

*   **Smart Comparison:** Upload two different resumes and paste your target Job Description. The tool evaluates which resume is the better match.
*   **Fallback Generation:** If both resumes fail to meet the required match threshold, the tool extracts your combined data to create a new, highly tailored resume.
*   **Industry Standard Formatting:** Automatically generates the new resume using the widely respected and ATS-friendly **Jake's Resume** template design.
*   **Live PDF Preview:** Review the newly generated resume directly in your browser.
*   **One-Click Download:** Download the final, perfectly tailored PDF instantly.
*   **Zero Setup:** Runs completely in the browser with no backend installation required.

## 🚀 How to Use

Since this is a lightweight, frontend-only application, you can start using it immediately:

1.  **Visit the Live Site:** [https://sanketthakare5050.github.io/resume-analyzer/]
2.  **Upload Resumes:** Select and upload `Resume_A` and `Resume_B`.
3.  **Paste the JD:** Copy and paste the full Job Description into the text area.
4.  **Analyze:** Click the analyze button to see which resume performs better.
5.  **Preview & Download:** If a new resume is generated, preview the Jake's Template layout and click download to get your PDF.

Alternatively, you can clone this repository and simply open the `index.html` file in any modern web browser.
